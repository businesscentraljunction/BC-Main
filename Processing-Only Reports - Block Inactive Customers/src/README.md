## 💡 Overview

This example demonstrates how to use a **Processing-Only Report** to automatically block inactive customers in Dynamics 365 Business Central.

## 🔧 Features

* Identify inactive customers based on sales activity
* Automatically block customers with no activity
* Perform background data processing without output

## 📂 Project Structure

* src/report
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Run the report
5. Customers without recent activity will be blocked

## ⚙️ Key Notes

* Uses Processing-Only report (no layout/output)
* Suitable for background data operations
* Ideal for automation scenarios
* Helps maintain clean and active customer data
