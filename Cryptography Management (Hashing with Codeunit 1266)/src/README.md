## 💡 Overview

This example demonstrates how to use the Cryptography Management codeunit to generate hash values in Dynamics 365 Business Central.

## 🔧 Features

* Generate hash values using multiple algorithms
* Support for MD5, SHA1, SHA256, SHA384, and SHA512
* Secure data handling and validation
* No need for external libraries

## 📂 Project Structure

* src/page
* AL cryptography example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open the cryptography page
5. Enter input text
6. Generate and view hash outputs

## ⚙️ Key Notes

* Uses Codeunit 1266 (Cryptography Management)
* Hashing is a one-way process (cannot be reversed)
* Different algorithms provide different security levels
* Useful for data integrity and secure storage
