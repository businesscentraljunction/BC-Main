## 💡 Overview

This example demonstrates how to search custom table data using the **"Search Company Data"** feature in Dynamics 365 Business Central.

## 🔧 Features

* Enable custom tables for search
* Configure searchable fields
* Search data using "Tell Me"

## 📂 Project Structure

* src/table
* src/page (Card + List)
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Enable table in "Set up where to search"
5. Search using keywords
