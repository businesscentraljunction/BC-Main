## 💡 Overview

This example demonstrates how to use **Page Labels** to display clear and always-visible instructions on pages in Dynamics 365 Business Central.

## 🔧 Features

* Display static instructional messages on pages
* Provide clear guidance to users before data entry
* Keep important notes always visible
* Improve user understanding and reduce errors

## 📂 Project Structure

* src/pageextension
* AL UI enhancement example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open the target page
5. View instructional message displayed on screen

## ⚙️ Key Notes

* Page Labels are always visible (unlike Tooltips or InstructionalText)
* Used for section-level or page-level instructions
* Can be placed before/after groups or between fields
* Ideal for setup pages, wizards, and complex forms
* Helps prevent user mistakes and improves clarity

## 🧠 Concept Insight

* Best used for important messages users should not miss
* Enhances usability without additional logic
* Commonly used in configuration and onboarding scenarios
* Provides a clean and structured way to guide users
