## 💡 Overview

This example demonstrates how to skip the request page in reports and execute them directly in Dynamics 365 Business Central.

## 🔧 Features

* Execute report without request page
* Run reports automatically
* Use predefined filters and settings

## 📂 Project Structure

* src/report
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Run the report
5. Report executes directly without request page

## ⚙️ Key Notes

* Request page is skipped during execution
* Users cannot apply filters at runtime
* Suitable for automated or system-driven reports
* Use predefined filters in AL code

## ⚠️ Limitations

* No runtime filtering
* No output selection options
* No easy cancellation
* May confuse users expecting interaction
