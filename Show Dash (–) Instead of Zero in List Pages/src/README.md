## 💡 Overview

This example demonstrates how to display a dash ("–") instead of zero for FlowFields in List or ListPart pages in Dynamics 365 Business Central.

## 🔧 Features

* Replace zero values with dash for better readability
* Improve UI consistency with standard pages
* Control formatting of calculated fields

## 📂 Project Structure

* src/pageextension
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open the page and verify formatted values
