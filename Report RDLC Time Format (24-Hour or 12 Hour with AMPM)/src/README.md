## 💡 Overview

This example demonstrates how to format DateTime values in RDLC reports to display time in a customized format (e.g., 15.00.00 PM) in Dynamics 365 Business Central.

## 🔧 Features

* Format time in 24-hour or 12-hour format
* Display AM/PM suffix
* Replace default separators (":" to ".")
* Customize report output using RDLC expressions

## 📂 Project Structure

* src/report
* RDLC layout example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open RDLC layout in Report Builder
5. Apply formatting expression to textbox
6. Preview formatted time output

## ⚙️ Key Notes

* Uses RDLC expressions for formatting
* Supports custom time formats and separators
* Improves readability of report output
* Works on DateTime fields in dataset
