## 💡 Overview

This example demonstrates how to use the **InstructionalText** property to guide users with helpful messages and improve data entry experience in Dynamics 365 Business Central.

## 🔧 Features

* Display helper text inside fields and groups
* Improve data entry clarity and usability
* Reduce user errors and confusion
* Support instructional guidance in pages and request pages
* Enhance onboarding experience

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open the page or request page
5. Observe instructional messages on fields/groups

## ⚙️ Key Notes

* Works on Page, Request Page, Field, and Group levels
* Supported for Text, BigText, Code, and Guid data types
* Displays as a placeholder-style helper message
* Optional parameters:

  * Locked → Prevents translation
  * Comment → Helps translators understand context
  * MaxLength → Controls text length
* Improves overall user experience and reduces training effort
