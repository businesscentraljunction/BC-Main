## 💡 Overview

This example demonstrates how to surface the **Block Reason** field on the Item Card in Dynamics 365 Business Central to improve visibility and communication.

## 🔧 Features

* Display Block Reason field on Item Card
* Improve clarity for blocked items
* Support both no-code and AL-based approaches

## 📂 Project Structure

* src/pageextension
* AL extension example

## 🚀 How to Use

1. Open in VS Code (for AL approach)
2. Download symbols
3. Publish extension
4. Open Item Card
5. View and update Block Reason field

## ⚙️ Key Notes

* Block Reason field already exists in the table
* Can be added using Personalization or AL extension
* Helps teams understand why items are blocked
* Improves communication across users
