## 💡 Overview

This example demonstrates how to save user-defined filters and options in the report request page using the **SaveValues** property in Dynamics 365 Business Central.

## 🔧 Features

* Remember user-applied filters
* Save report options automatically
* Improve report usability and efficiency

## 📂 Project Structure

* src/report
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Run the report and apply filters
5. Reopen the report to see saved values

## ⚙️ Key Notes

* Works only on request pages
* Saves filters and options per user
* Helps avoid repetitive input
* Use carefully when dynamic filters are involved
