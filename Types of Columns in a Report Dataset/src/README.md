## 💡 Overview

This example explains the different types of columns that can be used in a report dataset in Dynamics 365 Business Central.

## 🔧 Features

* Use table fields as columns
* Add variables as columns
* Create calculated (expression) columns
* Add static captions in dataset

## 📂 Project Structure

* src/report
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Run the report and review dataset output

## ⚙️ Key Notes

* Columns can be created from fields, variables, expressions, or static text
* Helps structure dataset for reporting layouts
* Improves flexibility in report design
