## 💡 Overview

This example demonstrates how to apply custom filtering on a field in the report request page in Dynamics 365 Business Central.

## 🔧 Features

* Apply custom filter on request page field
* Restrict lookup values based on criteria
* Filter report data dynamically based on user selection

## 📂 Project Structure

* src/report
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Run the report
5. Select filtered value from request page
6. View filtered report output

## ⚙️ Key Notes

* Uses `TableRelation` with filter condition
* Lookup shows only filtered values (e.g., starting with specific text)
* Report dataset is filtered based on selected value
