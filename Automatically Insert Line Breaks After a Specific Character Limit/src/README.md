## 💡 Overview

This example demonstrates how to break long text into multiple lines after a fixed number of characters in Dynamics 365 Business Central.

## 🔧 Features

* Automatically split long text
* Improve readability of "Work Description"
* Format text before storing in field

## 📂 Project Structure

* src/tableextension
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Insert record and verify formatted text

## ⚙️ Key Notes

* Line length can be controlled (e.g., 45 characters)
* Improves readability for long descriptions
