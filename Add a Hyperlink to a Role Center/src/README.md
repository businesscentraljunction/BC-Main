## 💡 Overview

This example demonstrates how to add a button on the Role Center page that opens an external website using AL in Dynamics 365 Business Central.

## 🔧 Features

* Add custom action to Role Center
* Open external URL in browser
* Integrate external resources within Business Central

## 📂 Project Structure

* src/codeunit
* src/pageextension
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open Role Center (e.g., Business Manager)
5. Click the custom button to open URL

## ⚙️ Key Notes

* Uses `Hyperlink` system function
* Requires a codeunit to trigger action
* Role Center actions do not support direct triggers
* Useful for quick access to external tools or documentation
