## 💡 Overview

This example demonstrates how to use the **FileUploadAction** property to automatically trigger file processing when users upload files in Dynamics 365 Business Central.

## 🔧 Features

* Trigger actions automatically after file upload
* Simplify file handling logic
* Improve user experience with seamless uploads
* Support multiple file uploads
* Process file streams directly
