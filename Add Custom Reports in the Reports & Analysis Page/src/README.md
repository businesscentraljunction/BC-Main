## 💡 Overview

This example demonstrates how to add custom reports to the **Reports & Analysis (Role Explorer)** area by extending a Role Center in Dynamics 365 Business Central.

## 🔧 Features

* Add custom report actions to Role Center
* Create grouped report section in Reporting area
* Surface reports in Reports & Analysis page

## 📂 Project Structure

* src/pageextension
* AL extension example

## 🚀 How to Use

1. Open in VS Code
2. Download symbols
3. Publish extension
4. Open Role Center (e.g., Business Manager)
5. Navigate to Reports & Analysis
6. View custom report group and actions

## ⚙️ Key Notes

* Reports & Analysis page cannot be extended directly
* Extend Role Center instead
* Use actions to add report entries
* Actions automatically appear in Reports & Analysis
